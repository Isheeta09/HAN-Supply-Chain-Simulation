# HAN-Supply-Chain-Simulation
Educational Supply Chain Management Simulation for HAN University of Applied Sciences

An interactive Supply Chain Management simulation developed in Python and Streamlit for HAN University of Applied Sciences.

The application allows students to experience real-world supply chain decision-making by managing Purchasing, Operations, Sales, and Supply Chain departments while balancing profitability, service level, sustainability, inventory, lead time, and risk.
---
##Project Objective

The purpose of this simulation is to help students understand how decisions made in different business functions affect overall supply chain performance.

Players must:

- Select supplier strategies
- Manage inventory levels
- Respond to demand fluctuations
- Handle supply chain disruptions
- Improve sustainability performance
- Balance cost, service, and risk
---
## Technologies Used

- Python
- Streamlit
- Pandas
---
## Key Features

### Interactive Dashboard

Monitor real-time business performance through:

- Efficiency Score
- Revenue & Net Profit
- Inventory Value
- Service Level
- ESG Score
- Lead Time
- Supply Chain Risk

### Department-Based Decision Making

Students make strategic decisions across:

- Purchasing
- Operations
- Sales
- Supply Chain

Each decision directly impacts company KPIs.

### Dynamic Supply Chain Events

The simulation includes realistic business scenarios such as:

- Rotterdam Port Strike
- Demand Surge
- Sustainability Pressure
- Tariff Shock
- Market Volatility
- CEO Strategic Challenge

### Financial Analysis

Generate and review:
- Income Statements
- Balance Sheets
- Quarterly Performance Reports
- KPI Trends

### Instructor Mode

Special instructor controls allow:

- Locking and unlocking game sections
- Managing classroom activities
- Triggering global disruption events
- Monitoring student progress

## Installation

### Clone the repository

```bash
git clone https://github.com/Isheeta09/HAN-Supply-Chain-Simulation.git
cd HAN-Supply-Chain-Simulation
```
### Install dependencies

```bash
pip install streamlit pandas
```

### Run the application

```bash
streamlit run modified_code
```

The application will open automatically in your browser at:

```text
http://localhost:8501
```
